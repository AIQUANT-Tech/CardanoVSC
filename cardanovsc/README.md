# CardanoVSC Support for Visual Studio Code

CardanoVSC is a powerful Visual Studio Code extension that provides seamless support for Haskell and Plutus development. It is designed to enhance developer productivity by offering:

- **Advanced Syntax Highlighting**: Enhanced syntax highlighting for Haskell and Plutus code, making it easier to read and understand your code.

- **Intelligent Code Completion**: Speed up your development process with intelligent code completion suggestions.

- **Cardano API Integration**: Seamless integration with the Cardano API for real-time blockchain interaction, allowing you to fetch blockchain data directly within the IDE.

- **Quick Access to Cardano API**: Use the intuitive "Ctrl + Shift + P >> CardanoAPI" feature to quickly access Cardano API options directly from the IDE.

## Additional Features

- **Webview UI**: Interactive webview for an enhanced user experience, providing a more dynamic and interactive way to interact with the extension.

- **Command Palette Commands**: Execute Cardano-specific commands directly from the command palette, such as retrieving block details, address balances, and more.

- **Snippets**: Predefined code snippets for Haskell to speed up your coding process.

- **Customizable Token Colors**: Customize the editor's token color settings for a better coding experience.

## Installation

1. Open Visual Studio Code.
2. Go to the Extensions view by clicking on the Extensions icon in the Activity Bar on the side of the window or by pressing `Ctrl+Shift+X`.
3. Search for "CardanoVSC".
4. Click the Install button.

## Usage

### Commands

CardanoVSC provides several commands to interact with the Cardano blockchain:

- **Hello World**: A simple command to get started.
- **Get Block Details**: Retrieve details of a specific block.
- **Get Latest Block Details**: Retrieve details of the latest block.
- **Get Address Balance**: Check the balance of a specific address.
- **Get Pool Details**: Retrieve details of a specific pool.
- **Get Pool Stats**: Get statistics for a specific pool.
- **Get Pools List**: List all available pools.
- **Get Pools Expiring**: List pools that are expiring soon.
- **Get Pools Expired**: List pools that have already expired.
- **Get Asset Details**: Retrieve details of a specific asset.
- **Get Assets by PolicyId**: List assets associated with a specific policy ID.
- **Get Assets by Address**: List assets associated with a specific address.
- **Get Transaction Details**: Retrieve details of a specific transaction.
- **Get Transaction List by Address**: List transactions associated with a specific address.
- **Get Stake Key Details**: Retrieve details of a specific stake key.
- **Get Governance Action**: Retrieve details of a governance action.
- **Get dReps List**: List all dReps.
- **Get dRep Information**: Retrieve details of a specific dRep.
- **Get Committee Members**: List all committee members.
- **Get Committee Information**: Retrieve details of the committee.
- **Get CCMember Details**: Retrieve details of a specific committee member.
- **Get CCHot Details**: Retrieve details of a specific CCHot.
- **Get Network Protocol Details**: Retrieve details of the network protocol.
- **Get Network Details**: Retrieve details of the network.
- **Get Addresses Associated with Stake Key**: List addresses associated with a specific stake key.
- **CardanoScan API Integration**: Integrate with the CardanoScan API.

### Snippets

CardanoVSC includes snippets for Haskell to speed up your development process. Simply start typing a snippet prefix and press `Tab` to insert the snippet.

### Syntax Highlighting

Enjoy enhanced syntax highlighting for Plutus and Haskell files, making your code easier to read and understand.

## Configuration

CardanoVSC allows you to customize the editor's token color settings for a better coding experience. You can modify these settings in your `settings.json` file.
